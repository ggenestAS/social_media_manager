# prep-ai post types — deliberately empty

This brand launches **paid-first**: the first content lives under
`output/paid/campaigns/` as experiments. Organic post types are added here by
*graduation* — when a paid creative or experiment format proves itself, its
winning draft becomes `<type>/{spec.md, template.html}` (see the experiment
skill's graduation rule). Don't pre-invent a menu nobody has validated.
